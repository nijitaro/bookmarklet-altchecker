=AltChecker

* 画像オーバー時に画像のaltと画像サイズがツールチップで表示されます。
* 空altの画像には青枠、alt無しの画像には赤枠、altの末尾にスペースが入っているものは緑枠が付きます。
* 全ての画像にalt、または空altが入っていた場合「Hi-Ho!!」とアラートします。
* 赤枠、緑枠があった場合はその画像のsrcをアラートします。

==インストール

1. 適当に何かしらブックマークする
2. altchecker.jsのソースをコピー
3. ブックマークの編集でURLにペースト、保存。(名前も変えるといいです)

== 小技

  Safariのブックマークバーは、フォルダを除くブックマークが左からそれぞれCmd1〜9に対応しています。
  名前を「｜(パイプ)」などにして保存すると、区切り線としてブックマークバーがすっきりします。
  実際に使用する際はCmd+○で呼び出せば早いです。
